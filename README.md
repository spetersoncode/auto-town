# Auto-Town

A 2D town-building simulation tech demo built with Godot 4.5.1 and C#.

## Features

- **Worker Management**: Assign workers to different jobs (Lumberjack, Miner, Forager, Builder)
- **Resource Gathering**: Workers collect wood, stone, and food from resource nodes
- **Building Construction**: Construct buildings like houses, sawmills, farms, and mines
- **Production Buildings**: Workers produce resources at farms and other production facilities
- **Task System**: Automated task assignment and worker AI

## Getting Started

1. Open the project in Godot 4.5.1 or later
2. Ensure .NET support is enabled
3. Run the project (F5)

## Controls

- **Click workers** to select/deselect them
- **Number keys (1-4)** to assign jobs to selected workers
- **F keys (F1-F4)** to build buildings
- **ESC** to show help/controls
